# Lumina Cricket Pro 2D 🏏

A high-fidelity 2D cricket simulation built with Pygame.

## Features
- **Realistic 2D/3D Physics**: Ball bounce, gravity, and trajectory simulation.
- **Rich Aesthetics**: Particle effects, shadows, screen shake, and smooth animations.
- **Dynamic Scoring**: Score 4s and 6s based on timing and power.
- **Dual Support**: Fully functional 2D (Pygame) and 3D (Ursina) versions.

## How to Play (2D)
1. **Move**: Use `LEFT` and `RIGHT` arrow keys.
2. **Bowl**: Press `SPACE`.
3. **Bat**: Press `SPACE`.

## How to Play (3D)
1. **Move**: Use `LEFT/RIGHT` or `A/D`.
2. **Bowl**: Press `ENTER` to start, bowlers runs automatically.
3. **Bat**: Press `SPACE` to swing.
4. **Camera**: Use mouse to look (optional) or automatic follow.

## Requirements
- Python 3.x
- Pygame-ce (`pip install pygame-ce`)
- Ursina (`pip install ursina`)

## Running the Game
### 2D Version
```bash
python cricket_2d.py
```
### 3D Version
```bash
python cricket_3d_pro.py
```
